Co-opvid



Co-opvid is a fun little trivia game all about COVID-19! Below you'll see a Game Creation menu, and below that lies the lobby, where you can join other multiplayer games. You can also make a singleplayer game.

The goal of each game is to fill in the missing letters of a word by answering COVID-related questions. If you get the question correct, one letter will go into the right spot. However, if you get it wrong, an incorrect letter will go into that spot. Each game has exactly 8 questions for you to enjoy. Have fun!
